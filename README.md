## Opinion App

Welcome to Opinion, an Opinion management system designed to help you track and manage your opinion efficiently. This project contains the full codebase for Opinion, organized into two main folders: backend and frontend. Each folder has its own README.md file with specific setup and run instructions.

## Project Scope

Backend: Contains the PHP codebase responsible for handling the server-side logic, including API endpoints, database interactions, and authentication. The backend is built using PHP and can be set up to run on a server or locally using tools like Docker.

Functional Requirments:

1. Admin can login & add post for opinion
2. Upload & manage Emoji's
3. Approved, reject comments
4. Manage personal profile
5. Share comments
6. Manage users

Frontend: Contains the Vue.js codebase responsible for the user interface, where users can input and track their expenses. The frontend is built using modern JavaScript technologies and is designed to provide a seamless user experience.

1. User can login & regrister
2. Add comments & reaction
3. Share post
4. See some statatic
5. See the comments after apptove.
6. Manage personal profile

## Prerequisites

Before setting up Expenso, ensure you have the following installed on your machine:

Node.js: Required for running the frontend.
PHP: Required for running the backend.
Composer: For managing PHP dependencies.
Docker: (Optional) If you prefer using Docker for a containerized setup.

Running the Project: After setting up both the backend and frontend, you can start the server and access the application through your browser. Ensure that the backend server is running before you launch the frontend.

## Run the following command:

```bash
docker-compose up --build

```
